Attention Pooling

Posted on 2025-01-27 Edited on 2025-01-28 In 深度学习 , LLM , Transformer , Attention
Attention 机制的核心思想—— Pooling，就是依据和 Query 的相关性，赋予不同的权重。

From Kernel Regression to Attention Pooling

Kernel Regression 是 Nadaraya-Watson 于 1964 年提出的对于 Regression 的改进，其中已经可以瞥见 Attention 的主要思想。

考虑一个非线性的回归问题 y=2sin⁡(x)+x0.8+ϵy=2\sin(x)+x^{0.8}+\epsilony=2sin(x)+x0.8+ϵ，如果我们只用简单的平均值的话，误差会变得非常得大。

Nadaraya 和 Watson 发现，这个问题是由于所有点的权重全部相同导致的，但是肯定是有些点更接近 Ground Truth，而有一些点远离，这样的话就不能用相同的权重了。所以他们提出了一种新方法，对点进行加权。这也是 Attention 的核心思想，为样本计算权重，和 Query 相关程度越大的权重越大。

我们来查看一下注意力权重，由于都是排序过的 test data 和 training data，发现主对角线上的颜色更深，说明 test data 的点对距离它更近的 training data 的点“越感兴趣”，因此权重越大，所以拟合的效果也越好。
